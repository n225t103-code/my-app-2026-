# 家計簿アプリ - Visual Analytics

シンプルで使いやすい、グラフ表示機能付きの家計簿アプリケーションです。ブラウザだけで動作し、入力したデータはローカルに安全に保存されます。

## 主な機能

- **収支管理**: 収入と支出を簡単に記録できます。
- **自動計算**: 残高、総収入、総支出をリアルタイムで計算して表示します。
- **グラフ可視化**: 支出の内訳をカテゴリ別にドーナツチャートで表示し、お金の使い道を一目で把握できます。
- **データ保存**: ブラウザの `localStorage` を使用するため、ページを閉じたり再読み込みしたりしてもデータは消えません。
- **モバイル対応**: スマートフォンやタブレットからも快適に操作できるレスポンシブデザイン。

## 使い方

1. **項目の追加**
   - 「項目を追加」フォームに、日付、収支タイプ（収入/支出）、カテゴリ、内容、金額を入力します。
   - 「保存する」ボタンを押すと、リストとグラフに反映されます。
2. **履歴の確認**
   - ページ下部の「履歴」セクションに、これまでの記録が新しい順に表示されます。
3. **項目の削除**
   - 履歴リストの右端にある「削除」ボタンを押すと、その項目を削除できます。削除後、合計金額とグラフも自動的に再計算されます。

## GitHub Pages での公開手順

このアプリを自分専用のウェブサイトとして公開する方法：

1. GitHubにリポジトリをプッシュします（完了済み）。
2. GitHubのリポジトリ画面で **Settings** タブをクリックします。
3. 左サイドバーの **Pages** を選択します。
4. **Build and deployment** セクションの **Branch** で `main` を選択し、右側の **Save** ボタンを押します。
5. 数分後、画面上部に公開URL（`https://<ユーザー名>.github.io/<リポジトリ名>/`）が表示されます。

## 技術スタック

- **HTML5 / CSS3**: レイアウトとデザイン
- **JavaScript (Vanilla JS)**: アプリケーションロジック
- **Chart.js**: グラフの描画
- **localStorage API**: データの永続化

## 注意事項

- データは使用しているブラウザ内に保存されます。別のブラウザやシークレットモード、別の端末からは同じデータを見ることはできません。
- ブラウザのキャッシュや履歴を完全に消去（サイトデータを削除）すると、保存されたデータも削除される場合がありますのでご注意ください。
